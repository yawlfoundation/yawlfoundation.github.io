User Manual

The comprehensive YAWL User Manual contains detailed descriptions on how to use the YAWL system, including the process editor, default work lists and administration pages.

Technical Manual

The YAWL Technical Manual contains detailed technical descriptions on the inner workings of the YAWL system and explanations on how developers can extend the system in numerous ways, including developing custom services and gateways, developing codelets, extending the resourcing service, creating custom forms and external data sources, and so on.
