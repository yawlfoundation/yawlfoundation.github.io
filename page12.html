Add Ons

The YAWL-Dashboard provides project management information for YAWL. If you want to install it on your machine, go here and download the zip file by clicking on the green Code button. Unpack the zip file and go to YALW-Dashboard-main/latest-build Follow the instructions in the README file in this directory. The Workflow Petri Net Designer WoPeD can translate workflow nets in PNML into YAWL nets. After downloading WoPeD Version 3.9 or later go to settings / YAWL and activate YAWL support. You will then find the option to save your WoPeD specification as a .yawl file. You can test this feature with LoanApplicationResources that you find in the example nets from the WoPeD menu.
